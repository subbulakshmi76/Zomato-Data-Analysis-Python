# Zomato Data Analysis Using Python

## Project Overview

This project analyzes restaurant data using Python and performs Exploratory Data Analysis (EDA) to gain insights into restaurant ratings, costs, and city-wise distribution.

## Objectives

* Analyze restaurant ratings and costs
* Identify top-rated restaurants
* Compare restaurant costs
* Visualize data using charts
* Understand city-wise restaurant distribution

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Dataset

A sample restaurant dataset was created and analyzed for this project.

## Analysis Performed

1. Data Creation and Loading
2. Data Exploration
3. Rating Analysis
4. Cost Analysis
5. Data Visualization

## Visualizations

* Restaurant Ratings Bar Chart
* Restaurant Cost Comparison Bar Chart
* Restaurants by City Pie Chart

## Key Insights

* A2B had the highest rating among the sample restaurants.
* Pizza Hut had the highest cost.
* Chennai contained the highest number of restaurants in the dataset.
* Visualizations helped identify trends in ratings and costs.

## Learning Outcomes

* Data Analysis using Pandas
* Data Visualization using Matplotlib
* Exploratory Data Analysis (EDA)
* Working with Jupyter Notebook
